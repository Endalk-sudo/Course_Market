# **Course Market**

Course Market is a modern, responsive React application designed to facilitate online course sales. This platform provides an intuitive interface for course creators to showcase their educational content and for students to browse and purchase courses. The application features a sleek, contemporary design with seamless navigation and a user-friendly experience.

## Features

- Responsive design that works on all devices
- Modern UI with clean aesthetics
- User-friendly course browsing
- Intuitive course purchasing flow
- Efficient course management

## Technologies Used

This project leverages the following technologies:

- **React** (^19.2.0) - A JavaScript library for building user interfaces
- **React DOM** (^19.2.0) - Provides DOM-specific methods for React
- **React Router DOM** (^7.9.6) - Declarative routing for React applications
- **React Icons** (^5.5.0) - Popular iconic fonts for React projects
- **Vite** (^7.2.4) - Next-generation frontend tooling for faster development
- **ESLint** (^9.39.1) - Pluggable JavaScript linter for code quality

## Installation

Follow these steps to get the project running locally:

1. Clone the repository:
   ```bash
   git clone "https://github.com/Endalk-sudo/Course_Market.git"
   ```

2. Navigate to the project directory:
   ```bash
   cd Course_Market
   ```

3. Install dependencies using npm:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

Your application will be running at [http://localhost:5173](http://localhost:5173) (or another port if 5173 is occupied).
